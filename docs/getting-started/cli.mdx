# Panda CLI

The simplest and fastest way to use Panda is with the Panda CLI tool.

## Install Panda

```bash
npm install -g @pandacss/dev
npx panda init
```

## Configure the content

```js {3}
import { defineConfig } from '@pandacss/dev'

export default defineConfig({
  preflight: true,
  presets: ['@pandacss/dev/presets'],
  // define the content to scan ğŸ‘‡ğŸ»
  include: ['./src/**/*.{tsx,jsx}', './pages/**/*.{jsx,tsx}'],
  exclude: [],
  outdir: 'styled-system',
})
```

## Configure the entry CSS with layers

```css
@layer reset, base, tokens, recipes, utilities;
```

## Start the Panda extract process

Run the CLI tool to scan your component files for style properties and css call expressions.

```bash
# Run it once
panda

# Run it in watch mode
panda --watch
```

## Start using Panda in your code

Use the generated style utilities in your code and panda will extract them to the generated CSS file.

```jsx
import { css } from './styled-system/css'

export function App() {
  return <div className={css({ bg: 'red.400' })} />
}
```
