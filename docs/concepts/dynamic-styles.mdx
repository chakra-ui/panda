# Writing dynamic styles

Due to the fact that Panda extracts styles statically, it is unable to resolve dynamic styles.

```js
// ‚ùå This won't work
css({ background: isActive ? 'red' : 'blue' })
```

## Using css map (recommended)

The `cssMap` function creates a collection of named style objects.

This is useful for mapping component props to styles

```js
const background = cssMap({
  primary: { backgroundColor: 'blue' },
  secondary: { backgroundColor: 'red' },
})

const bg = background(props.bg)
```

## Using css variables

Leveraging css variables is also a great way to provide dynamic values to css properties.

```js
<button
  className={css({
    background: 'var(--bg-color)',
  })}
  style={{
    '--bg-color': isActive ? 'red' : 'blue',
  }}
>
  Click me
</button>
```
