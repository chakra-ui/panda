# Get started with Panda

Panda combines the developer experience of CSS-in-JS and the performance of utility css. It scans your JavaScript and TypeScript files for style props and function calls, generating the corresponding styles and then writing them to a static CSS file.

## Installation

- [Panda CLI](/docs/getting-started/cli.mdx):
  The simplest and fastest way to use Panda is with the Panda CLI tool.

- [Using PostCSS](/docs/getting-started/postcss.mdx):
  Installing Panda as a PostCSS plugin is the most seamless way to integrate it with build tools like webpack, Rollup, Vite, and Parcel.

- [Using Astro](/docs/getting-started/astro.mdx):
  [Astro](https://astro.build/) is an all-in-one web framework for building fast, content-focused websites.

- [Using Next.js](/docs/getting-started/nextjs.mdx):
  [Next.js](https://nextjs.org/) is a React framework for building full-stack Web applications.

- [Using Vite](/docs/getting-started/vite.mdx):
  [Vite](https://vitejs.dev/) is a fast build tool for modern web projects.
