# Using Panda CSS with Next.js

Panda CSS works out of the box with Next.js. You can use Panda CSS with Next.js in two ways:

- [Using Panda CSS PostCSS plugin](#panda-css-setup-using-postcss-plugin)
- [Using Panda CSS CLI](#panda-css-setup-using-panda-css-cli)

If you don't have Next.js app installed, you can follow the [Create a Next.js app](#create-a-nextjs-app) section to create a new Next.js app, otherwise you can skip to the [Install Panda CSS](#install-panda-css) section.

## Create a Next.js app

First, create a Next.js app using the official [Create Next App](https://nextjs.org/docs/api-reference/create-next-app) CLI.
We will name our project `my-panda-css-app` but you can name it whatever you want.

### Interactive

If you don't enter any parameter, the CLI will guide you through the process of creating a new Next.js app.

## Panda CSS setup using PostCSS plugin

### Init Panda CSS

Run the following command to initialize Panda CSS in your project:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm panda init --postcss
    ```
  </Tab>
  <Tab>
    ```bash
    npm exec panda init --postcss
    ```
  </Tab>
  <Tab>
    ```bash
    yarn panda init --postcss
    ```
  </Tab>
</Tabs>

### Update package.json scripts

Open your `package.json` file and update the `scripts` section as follows:

```diff
{
  "scripts": {
+    "prepare": "panda codegen",
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```
- `"prepare"` - script that will run Panda CSS CLI codegen before each build. Read more about [codegen](/docs/references/cli#panda-codegen) in the CLI section.

### Add panda CSS styles to your app layout

In your Next.js project, navigate to the `src/app` folder and open `global.css` file. Replace all the content with the following code:

```css
@layer reset, base, tokens, recipes, utilities;
```

### PostCSS config

`panda init --postcss` command will automatically create a `postcss.config.js` file in the root of your project with the following code:

```js
module.exports = {
  plugins: {
    '@pandacss/dev/postcss': {},
  },
}
```

For advanced configuration follow the Next.js PostCSS guide to set up a custom PostCSS configuration by referring to this [link](https://nextjs.org/docs/pages/building-your-application/configuring/post-css#customizing-plugins).

### Known issues

Sometimes Next.js caches PostCSS generated styles and you need to clear the cache. To do that, delete the `.next` folder and restart your development server.

You can also update you `package.json` scripts to delete the `.next` folder before each build:

```diff
{
  "scripts": {
-    "dev": "next dev",
+    "dev": "rm -rf .next && next dev",
  },
}
```

## Panda CSS setup using Panda CSS CLI

### Update package.json scripts

Open your `package.json` file and update the `scripts` section as follows:

```diff
{
  "scripts": {
+    "prepare": "panda",
-    "dev": "next dev",
+    "dev": "concurrently 'npm:dev:*'",
+    "dev:css": "panda --watch",
+    "dev:next": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```
- `"prepare"` - script that will run Panda CSS CLI before each build. The difference is that this script will output Panda CSS styles `./styled-system/styles.css` that we can then inport in our `layout.tsx`.
- `"dev"` - script that will run both `dev:css` and `dev:next` scripts concurrently.
- `"dev:css"` - script that will watch for changes in your `panda.config.ts` and automatically rebuild them during development.
- `"dev:next"` - script that will run Next.js development server.

You will also need to install the `concurrently` package as a dev dependency to run multiple npm scripts concurrently.

You can install it using npm with the following command:

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm install concurrently --save-dev
    ```
  </Tab>
  <Tab>
    ```bash
    npm install concurrently --save-dev
    ```
  </Tab>
  <Tab>
    ```bash
    yarn install concurrently --save-dev
    ```
  </Tab>
</Tabs>

### Update Next.js `layout.tsx` and index page `page.tsx`:

In your Next.js project, navigate to the `layout.tsx` file located at `src/app/layout.tsx`. Replace the import statement for `./globals.css` with `import ../../styled-system/styles.css`.

The final snippet for `layout.tsx` would look like this:

```tsx
import '../../styled-system/styles.css';
import { Inter } from 'next/font/google'

const inter = Inter({ subsets: ['latin'] })

export const metadata = {
  title: 'Create Next App',
  description: 'Generated by create next app',
}

export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="en">
      <body className={inter.className}>{children}</body>
    </html>
  )
}
```

## Write your first Panda CSS styles

We will update the contents of `page.tsx` with the following snippet that uses Panda CSS:

```tsx
import { css } from '../../styled-system/css';
```
