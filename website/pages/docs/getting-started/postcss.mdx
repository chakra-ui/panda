# Using PostCSS

Installing Panda as a PostCSS plugin is the most seamless way to integrate it with build tools like webpack, Rollup,
Vite, and Parcel.

## Install Panda

Install panda and create your `panda.config.ts` file.

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm install -g @pandacss/dev postcss autoprefixer
    pnpx panda init --postcss
    ```
  </Tab>
  <Tab>
    ```bash
    npm install -g @pandacss/dev postcss autoprefixer
    npx panda init --postcss
    ```
  </Tab>
  <Tab>
    ```bash
    yarn install -g @pandacss/dev postcss autoprefixer
    npx panda init --postcss
    ```
  </Tab>
</Tabs>

## Add Panda to your PostCSS config

Add panda and autoprefixer to your `postcss.config.cjs` file, or wherever PostCSS is configured in your project.

```js
module.exports = {
  plugins: {
    autoprefixer: {},
    '@pandacss/dev/postcss': {}
  }
}
```

## Configure the content

```js {3}
import { defineConfig } from '@pandacss/dev'

export default defineConfig({
  preflight: true,
  presets: ['@pandacss/dev/presets'],
  // define the content to scan üëáüèª
  include: ['./src/**/*.{tsx,jsx}', './pages/**/*.{jsx,tsx}'],
  exclude: [],
  outdir: 'styled-system'
})
```

## Configure the entry CSS with layers

```css
@layer reset, base, tokens, recipes, utilities;
```

## Start your build process

Run your build process with `npm run dev` or whatever command is configured in your package.json file.

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm run dev
    ```
  </Tab>
  <Tab>
    ```bash
    npm run dev
    ```
  </Tab>
  <Tab>
    ```bash
    yarn run dev
    ```
  </Tab>
</Tabs>

## Start using Panda in your code

Use the generated style utilities in your code and panda will extract them to the generated CSS file.

```jsx
import { css } from './styled-system/css'

export function App() {
  return <div className={css({ bg: 'red.400' })} />
}
```
