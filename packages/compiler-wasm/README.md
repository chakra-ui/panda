# @pandacss/compiler-wasm

The WebAssembly build of the Panda CSS compiler engine, for browser and playground targets where the native
`@pandacss/compiler` binding can't run.

This package is consumed by Panda tooling; you typically don't install it directly.

## Documentation

Visit the [Panda CSS documentation](https://panda-css.com) to learn more.
