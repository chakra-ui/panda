# @pandacss/mcp

A [Model Context Protocol](https://modelcontextprotocol.io) server for [Panda CSS](https://panda-css.com), giving AI
assistants structured access to the Panda compiler.

## Usage

```bash
npx -y @pandacss/mcp
```

## Documentation

Visit the [Panda CSS documentation](https://panda-css.com) to learn more.
