# @pandacss/preset-base

The base preset for [Panda CSS](https://panda-css.com) — the built-in conditions and utilities. Included by default in
every Panda project.

## Installation

```bash
npm install -D @pandacss/preset-base
```

## Documentation

Visit the [Panda CSS documentation](https://panda-css.com) to learn more.
