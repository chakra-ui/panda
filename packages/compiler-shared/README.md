# @pandacss/compiler-shared

Shared TypeScript contract and glue for the Panda compiler bindings (the native `@pandacss/compiler` and the
`@pandacss/compiler-wasm` build). Internal package; you typically don't install it directly.

## Documentation

Visit the [Panda CSS documentation](https://panda-css.com) to learn more.
