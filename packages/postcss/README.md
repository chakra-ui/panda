# @pandacss/postcss

The PostCSS plugin for [Panda CSS](https://panda-css.com).

## Installation

```bash
npm install -D @pandacss/postcss
```

> If you use `@pandacss/dev`, you can reference the plugin as `@pandacss/dev/postcss` without installing this package
> separately.

## Usage

```js
// postcss.config.cjs
module.exports = {
  plugins: {
    '@pandacss/postcss': {},
  },
}
```
